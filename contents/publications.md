- 黑龙江省哲社科项目 《智能信息技术驱动的我国地方外宣话语跨文化适配框架》             2025.12.09-至今
内容：目标为训练用于审核黑龙江省地方官方外宣文本的AI智能体，推进政府行政数智化进程。目前与计算学部合作，尝试构建将语言学规则转化为参数化指令的理论框架与技术路径，以此来将跨文化交际规则具象化。
贡献：主动担任数据工程负责人，基于已完成的理论成果设计了产品demo，以完成面向计算学部同学的项目可行性和进一步需求的说明。具体为对GitHub上开源Claude Code 前端工程进行二次开发，为地方政府机关设计英文新闻撰写与润色 AI Agent 整合包。通过 Docker 构建虚拟办公沙箱隔离敏感文件，以 Skill 形式注入自研的 AIC-CA 英文新闻质量评价体系作为自动加载上下文，实现稿件上传后自动量化评分与修改建议生成。针对政府客户重新设计 UI 界面，并对 Claude Code 进行权限收敛与 API Key 加密脱敏，初步消除硬编码密钥泄露风险。经过文本测试，注入skill的智能体生成的新闻文本与同题材的英文母语文本的风格拟合度相比于注入之前平均提升22.51%且具有显著性。
